# Carlthuringer Com
Website production engine for DAT

## SETUP
* NVM: `curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.33.11/install.sh | bash`
* NPM: `npm install`

## Development
* Build: `npm run build`
* Publish: `npm run publish`
  Wait a few seconds for hashbase to pick up the new files.
